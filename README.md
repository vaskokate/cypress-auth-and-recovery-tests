# cypress-auth-and-recovery-tests

Учебная практика написание автотестов для проверки авторизации и восстановления пароля

Позитивные кейсы:
1. Проверка работы полей ввода: возможность ввода и удаления текста.
2. Авторизация с корректными логином и паролем.
3. Нажатие кнопки «Войти».
4. Подтверждение успешной авторизации.
5. Проверка отображения сообщения при неудачной авторизации (общий кейс).

Негативные кейсы:
1. Корректный логин + некорректный пароль.
2. Некорректный логин + корректный пароль.
3. Проверка валидации логина (отсутствие символа @).

Тест восстановления пароля:
1. Переход в форму восстановления (кнопка «Забыли пароль»).
2. Ввод email в поле восстановления.
3. Проверка отображения ожидаемого текстового сообщения.
4. Проверка наличия кнопки закрытия (крестика).

Специальный тест (проверка бага): проверка автоматического приведения логина к нижнему регистру.
Примечание: тест должен завершиться с ошибкой — это подтверждает наличие бага
